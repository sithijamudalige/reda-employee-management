# REDA Employee Management System

A Django-based employee management system for tracking attendance, managing member records, and generating reports.

## Features

- ✅ Member Management (CRUD operations)
- ✅ Attendance Tracking with leave dates
- ✅ Advanced Filtering (Province, Branch, Gender, Position Category)
- ✅ PDF Report Generation
- ✅ CSV/Excel Export
- ✅ Workplace Assignment History
- ✅ Service Years Calculation
- ✅ Responsive UI with Bootstrap
